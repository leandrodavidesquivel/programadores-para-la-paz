Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente JSON:

{
"nombre": "Maria",
"mensaje": "Hola comunidad"
}

el servidor recibe los datos enviados en el cuerpo de la petición mediante req.body y responde con un objeto JSON que confirma la recepción de la información:

{
"estado": "Datos recibidos",
"nombre": "Maria",
"mensaje": "Hola comunidad"
}

Esta respuesta indica que el servidor procesó correctamente los datos enviados.
