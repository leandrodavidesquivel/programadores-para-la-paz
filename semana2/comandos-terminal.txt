Los comandos que usado para la creacion de esta actividad son los siguientes:
cd -  para moverme en cada ubicacion.
pwd - para saber mi ubicacion.
ls - para saber que hay dentro de mi ubicacion, dentro de la carpeta actual.
echo - para imprimir textos.
nano - para editar el texto desde la terminal.
touch - para crear archivos.
mkdir - para la creacion de carpetas.
cat - para ver el contenido.
rm - para eliminar archivos.
clear - limpiar mi terminal.
